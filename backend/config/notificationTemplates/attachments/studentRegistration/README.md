Place the PDF you want to send with student registration emails in this folder.

How it works:
- Any `.pdf` file placed here can be attached to the student welcome email.
- If there are multiple PDFs, the most recently modified one is used.
- If this folder has no PDF, the system falls back to the auto-generated setup PDF.

Recommended:
- Keep only one PDF here to avoid confusion.
- Example filename: `student-profile-setup.pdf`
